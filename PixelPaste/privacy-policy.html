Privacy Policy

Transparency is our priority. PixelPaste is built to respect your data.

1. Overview

PixelPaste ("the Extension") is developed by askharoun.dev. We believe that your clipboard data and images belong to you. This Privacy Policy explains how the Extension handles your data.

2. Data Collection

We do not collect, store, or transmit any personal data.

The Extension does not have a backend server. It does not use analytics software (such as Google Analytics) to track user behavior within the extension popup or background scripts.

3. Local Processing

All image processing happens locally on your device:

  • Clipboard Data: When you paste an image, it is read directly from your system clipboard into the extension's local memory to generate a preview.
  • File Saving: When you click "Save", the file is generated using your browser's Canvas API and saved directly to your local downloads folder.
  • No Uploads: No image data is ever sent to an external server.

4. Permissions Usage

The Extension requests specific permissions for functionality only:

  • activeTab: Required to access the current tab only when you explicitly click the extension icon. This allows the extension to function without requiring broad access to your browsing history.
  • scripting: Required solely for the "Picture-in-Picture" feature. This allows the extension to inject a temporary HTML overlay (the floating image) into the web page you are currently viewing.
  • Host Permissions (all_urls): Required to ensure the Picture-in-Picture feature works on any website you visit.
