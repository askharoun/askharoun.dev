PixelPaste – Instant Clipboard to File

A Chrome extension to instantly paste clipboard images, rename them, and save as PNG or JPG. 100% Local processing with floating PiP mode.

Available on the Chrome Web Store
Open PixelPaste on Chrome Web Store
Click to install directly in Chrome

Workflow Evolved

I built PixelPaste because I was tired of opening Photoshop just to save a screenshot as a specific file. It streamlines the process of capturing and organizing visual assets.

  • Instant Capture: Just hit Ctrl+V. The image appears instantly.
  • Smart Renaming: Type a filename and hit Enter to save.
  • Dual Formats: Toggle between PNG (lossless) and JPG (smaller size).
  • 100% Privacy: Runs entirely offline in your browser.

Tech stack and permissions

Built with modern web technologies focused on performance and privacy.

Tech stack Manifest V3, JavaScript, Canvas API
Permissions activeTab, scripting (for PiP function)
Privacy Focus No analytics, no external server calls
Role Full Stack Developer & Designer

Picture-in-Picture Mode

Need to reference a diagram while coding or writing? PixelPaste includes a unique "Float" feature that injects your image into the current webpage as a draggable overlay.

The floating window stays on top of your work until you close it, acting like a seamless "second monitor" experience inside a single browser window.
